# Technical Director Knowledge

## Назначение

Помогает субагенту `technical-director` отвечать по паровым котлам,
комплектациям, нормативке, устройству котла и сдаче объектов в эксплуатацию.

## Источники

Используй только подключенные или предоставленные источники:

- NotebookLM/RAG экспорт или локально доступная база знаний
- нормативные документы
- Telegram/MAX технические чаты
- переписка с конструкторами
- спецификации и комплектации котлов
- акты, чеклисты и документы по вводу в эксплуатацию

Если NotebookLM не имеет прямого API в текущем контуре, работай через экспорт,
локальные документы или отдельный RAG-коннектор.

## Процесс ответа

1. Определи тип вопроса: нормативка, конструкция, комплектация, эксплуатация, сдача.
2. Найди подтверждение в источниках.
3. Укажи, что подтверждено документом, а что является инженерной гипотезой.
4. Если вопрос влияет на договор, запроси `legal`.
5. Если вопрос влияет на сроки производства, запроси `production`.
