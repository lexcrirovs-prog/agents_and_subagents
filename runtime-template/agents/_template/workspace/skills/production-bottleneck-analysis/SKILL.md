# Production Bottleneck Analysis

## Назначение

Помогает субагенту `production` анализировать состояние производства котлов,
склад и ограничения потока.

## Входные данные

Предпочтительный файл: CSV/XLSX с полями:

- номер котла или заказа
- модель/тип
- текущий этап
- дата входа на этап
- плановая дата выхода
- фактическая дата выхода
- ответственный участок
- статус комплектации
- складской статус
- блокер

## Анализ

1. Посчитай длительность каждого этапа.
2. Найди этапы с максимальным средним ожиданием.
3. Отдели нехватку материалов от нехватки людей, оборудования и согласований.
4. Проверь, какие котлы готовы на складе и какие зависли в WIP.
5. Сформируй 3-5 действий, которые быстрее всего ускорят поток.

## Формат вывода

```text
Состояние производства:
- ...

Узкие места:
- участок / причина / эффект на срок

Быстрые действия:
- ...

Какие данные нужны точнее:
- ...
```

Если файл неполный, не додумывай цифры; дай список недостающих колонок.
