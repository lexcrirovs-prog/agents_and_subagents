# Sales Script Example

## Stages

1. Greeting and company identification.
2. Clarify customer need and boiler context.
3. Ask about timing, budget, object status and decision maker.
4. Present relevant solution and next step.
5. Handle objections.
6. Confirm next action and deadline.

## Scoring

- 0: missing
- 1: weak or incomplete
- 2: acceptable
- 3: strong

## Required Report Fields

- manager
- call date
- customer context
- script score
- strongest moment
- main missed opportunity
- recommendation
